# WuXi Biologics — Record 2025 Annual Results

บทความสรุปผลประกอบการปี 2025 ของ WuXi Biologics (2269.HK)

## 🚀 Deploy to GitHub Pages

### วิธีที่ 1: ผ่าน GitHub.com (ง่ายที่สุด)

1. ไปที่ [github.com/new](https://github.com/new) สร้าง Repository ใหม่
2. ตั้งชื่อ เช่น `wuxi-biologics-2025`
3. เลือก **Public**
4. กด **Create repository**
5. กด **"uploading an existing file"**
6. ลาก `index.html` ไปวาง → กด **Commit changes**
7. ไปที่ **Settings → Pages**
8. ตั้ง Source เป็น **Deploy from a branch**
9. เลือก Branch: `main` / Folder: `/ (root)` → กด **Save**
10. รอ 1-2 นาที เว็บจะพร้อมใช้งานที่:

```
https://<your-username>.github.io/wuxi-biologics-2025/
```

### วิธีที่ 2: ผ่าน Git Command Line

```bash
# 1. สร้าง repo บน GitHub ก่อน แล้วรันคำสั่งนี้
git init
git add .
git commit -m "Initial commit: WuXi Biologics 2025 article"
git branch -M main
git remote add origin https://github.com/<YOUR_USERNAME>/wuxi-biologics-2025.git
git push -u origin main

# 2. เปิด GitHub Pages ใน Settings → Pages → Branch: main → Save
```
